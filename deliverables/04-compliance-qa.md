# 脚本合规与可拍摄性质检

## 事实与广告风险

| 检查项 | 结果 | 说明 |
|---|---|---|
| 减肥/减脂结果承诺 | 通过 | 未出现减重、掉秤、燃脂等表达 |
| 降糖或医疗功效 | 通过 | 仅按 Brief 使用“0蔗糖”，未写成无糖或降糖 |
| 高蛋白依据 | 条件通过 | 必须由最终产品标签或品牌材料确认 |
| 绝对化表达 | 通过 | 未使用必然、保证、零负担等用语 |
| 身材焦虑 | 通过 | 无体重、腰围或羞辱性对比 |
| 商业披露 | 通过 | 分镜包含品牌合作披露占位 |

## 可拍摄性

| 检查项 | 结果 | 说明 |
|---|---|---|
| 单人拍摄 | 通过 | 手机三脚架即可完成 |
| 场地与道具 | 通过 | 厨房、门口及常见食材 |
| 连续性 | 通过 | 蓝莓味为主杯，三口味统一补拍 |
| 产品可见度 | 通过 | 包装、质地、食用和三口味均有镜头 |
| 时长 | 通过 | 目标38秒，预留约2秒剪辑弹性 |

## 拍摄前仍需品牌确认

- 最终包装是否支持“0蔗糖”和“高蛋白”。
- 三个口味的准确商品名及包装版本。
- 是否需要固定披露文案、Logo 或购买 CTA。
- 是否允许展示麦片和水果等非品牌配料。
