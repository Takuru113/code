# 食品商单合规检查表

## 出现即退回

- 减肥、瘦身、减脂结果承诺或体重变化保证。
- 降糖、降血糖、治疗、预防疾病等医疗或保健功效。
- “零负担”“随便吃”“怎么吃都不胖”等绝对化表达。
- 将个人体验包装成普遍有效结论。
- 没有标签或品牌证据支撑的蛋白质、糖、热量数字。
- 用身材羞辱或焦虑迫使用户购买。

## 必须改写或补证据

- 把饱腹感写成持续时长或必然结果。
- 把 0 蔗糖偷换成无糖或控糖食品。
- 暗示运动恢复、增肌等生理功效。
- 对孕期、哺乳期或疾病人群给出食用建议。

## 推荐表达

- “我早上会搭配……”
- “这一杯标签写的是高蛋白、0蔗糖。”
- “质地更浓稠，搭水果或麦片很方便。”
- “忙的时候省一步准备，口味按自己喜欢选。”

## 可拍摄性

确保单人能完成动作和机位，食物状态与包装朝向连续，不依赖专业器材，且台词按普通语速不超时。
