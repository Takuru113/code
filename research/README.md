# 达人调研说明

本目录保存“小红书达人调研 + 商单脚本生成助手”的调研过程与证据索引。

## 核实规则

1. 账号名称、小红书号、粉丝量级和近期内容必须来自核实当日可访问的小红书页面。
2. 第三方平台仅用于发现候选人和交叉验证，不替代小红书站内核实。
3. 粉丝数若页面只显示“1万+”等区间，原样记录，不反推精确数字。
4. “粉丝画像”属于基于内容主题、评论语境和账号人设的推断，必须明确标为推断。
5. 未能打开主页或无法确认身份的账号不进入最终 10 人名单。
6. 近期内容以核实当日主页可见内容为准，记录标题和页面链接；不虚构发布时间或互动数。

## 候选筛选维度

- 人群匹配：22–35 岁城市女性相关度。
- 内容匹配：健身、控糖、轻食、早餐、运动后补给、上班族效率生活。
- 场景适配：能否自然出现酸奶，并适合真人短视频拍摄。
- 表达适配：是否具备清晰钩子、生活化表达和自然种草能力。
- 商业风险：是否长期依赖夸张减重、医疗化或绝对化承诺。
- 可验证性：主页、账号标识、近期内容和粉丝量级是否能够直接核实。

## 目录规划

- `candidate-pool.md`：候选发现、核实进度和淘汰理由。
- `talent-research.md`：最终 10 位达人调研表。
- `evidence/`：页面截图和逐账号证据说明。
- `sources.md`：平台规则、行业资料和第三方交叉验证来源。
