# Brief 与输出结构

## Brief 字段

```yaml
brand:
product:
supported_claims: []
forbidden_claims: []
target_audience: []
platform:
duration_seconds:
primary_scene:
creator:
creator_source_url:
creator_verified_at:
required_disclosure:
production_limits: []
```

## 稳定输出顺序

1. Brief 摘要
2. 事实、推断与创作建议
3. 达人风格抽象
4. 内容策略与核心钩子
5. 完整口播/对白脚本
6. 分镜表
7. 合规与可拍摄性检查
8. 待品牌确认字段

分镜必须包含时间、景别、画面与动作、台词/旁白、屏幕字、产品露出、道具/地点和声音。
