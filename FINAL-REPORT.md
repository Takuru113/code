# MCN 实操项目最终报告

## 1. 项目目标

为“轻醒”0蔗糖高蛋白希腊酸奶建立一套从真实达人调研到商单脚本、风险质检和飞书交付的 Agent 工作流，目标用户为关注健身、轻食、控糖语境和效率生活的 22–35 岁城市女性。

## 2. 研究方法

先使用公开搜索和第三方达人资料页建立候选池，再逐个进入小红书当前主页核实账号名称、小红书号、公开粉丝量级、简介和近期内容。第三方资料仅用于发现和交叉验证；若与站内不一致，以 2026-07-16 站内信息为准。

本轮检查超过 15 位候选。`上班了崽崽` 已转型为 AI/数码账号；`自律一只猫`、`大鹅睡醒了`、`你的饭搭子叫公主` 和 `21min9` 因快速减重、绝对化承诺或身材焦虑风险被排除。

## 3. 最终达人选择

最终 10 人及完整评分见 `research/talent-research.md`。第一名为 **Shirley爱吃爱练💪（93/100）**。

- **人设**：健身与高蛋白饮食并行，符合“爱吃也爱练”的轻负担生活方式。
- **场景**：主页已有 5 分钟高蛋白早餐、快速午餐和高蛋白食材内容。
- **表达**：常用数字、少材料、短时间作为钩子，适合小红书快节奏视频。
- **产品匹配**：早餐、运动后和下午茶均可自然出现酸奶。
- **风险可控**：需要去除既有内容中的减脂和体重结果表达，但不影响其核心结构。

## 4. 内容方案

内容母题为“早八健身女孩的5分钟高蛋白早餐”。视频从看时间和开冰箱切入，通过撕盖、质地、加蓝莓和麦片、直接食用、三口味蒙太奇和拿包出门完成 38 秒叙事。

方案没有使用体重、腰围、前后对比或减重承诺；产品价值通过真实动作体现。完整脚本和 8 镜分镜见 `deliverables/03-script-storyboard.md`。

## 5. Agent 方案

工作流共 5 步：Brief 拆解、真实达人核实与风格抽象、内容策略与脚本生成、合规及可拍摄性质检、飞书结构化写入。

核心 Prompt 强制区分事实、推断和创作建议；缺失数据标记未知；0蔗糖不得改写为无糖；高蛋白和任何数字必须有产品材料支持。可复用 Skill 位于 `skills/mcn-script-assistant/`。

## 6. 飞书自动化

飞书写入器使用 Python 标准库实现，默认 dry-run。真实执行时从环境变量读取凭据，换取 `tenant_access_token`，创建新版文档，批量追加块，并通过元数据和纯文本接口回读验证。程序对接口频控和临时服务错误执行指数退避。

脱机验证结果：5 份核心交付文档转换为 206 个文本块、8010 个字符，程序语法检查和 dry-run 均通过。真实写入需要用户完成一次应用创建和授权。

## 7. 合规结论

当前脚本未发现未解决的高风险项目。以下两项为拍摄前条件：

- 最终包装或品牌证明必须支持“0蔗糖”和“高蛋白”。
- 品牌需确认合作披露文案、包装版本、CTA 和非品牌配料展示规则。
